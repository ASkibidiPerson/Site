❓ FAQ

What is Glazed? +
Free Meteor addon for DonutSMP with 78 modules. Automates boring tasks and provides high-quality tools for free.
Which versions work? +
Minecraft 1.21.4, 1.21.10, and 1.21.11.
Is it really free? +
Yes — completely open source. No paid version exists.
Will I get banned? +
v16.1 includes bypass for Donut autobans. Use at your own risk.
How to install? +
Download .jar from GitHub releases, place in Meteor Client's 'addons' folder, launch Minecraft.
Best modules to start? +
Auto Tree Farmer, AH Sniper, Premium Tunnel Base Finder, Aim Assist — pair with Auto Replenish & Auto Eat.
